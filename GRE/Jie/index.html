Projects

GRE (Graph Runtime Engine)

September 2010 - Present

GRE is a platform for emerging large-scale distributed graph-parallel computation on real-world graphs with over billions of vertices and edges. Essentials of GRE are our new proposed abstractions on both computation model and distributed graph model, as well as an efficient underlying runtime highly optimized for both multi-core and network communication. It runs 2.5-17X faster than PowerGraph (GraphLab 2.2) for applications such as PageRank and Single Source Shortest Path.

Sweep-H

March 2011 - July 2011

Sweep-H is a framework for hybrid parallel Sn Sweeps on large-scale unstructured grids. Sweep-H solves the Sn Sweep problem in a way of data-driven traversal on directed acyclic graphs. Experiments with real-world grids on a cluster with 64 machines (768 cores) demonstrate Sweep-H's near linear scalability and significant performance advantage over traditional designs.

SMART

September 2008 - April 2010

SMART, the Sampling, Monitoring, Analyzing, and ReTuning, is a tool suite for multi-threading parallel programs. It was part of National Projects for 'nucleus electronic components, high-end general processors, and fundamental software'. I proposed the software architecture with my supervisor, and developed the Linux kernel extension (its design is similar to today's perf) component and a pseudo-device module.
